# ClawChat2 v0.2.2 Release Notes

## English

This release continues the current ClawChat2 Android baseline and prepares the next public build.

ClawChat2 is an unofficial OpenClaw Android fork focused on a simple, direct, chat-first experience for talking with OpenClaw agents.

Highlights in this build:

- `openclaw-webchat` is now the main source of truth for contacts, sessions, and chat history
- contacts UI has been refined with synced agent avatars and cleaner contact-row presentation
- agent-sent media support remains a core focus, including stable image, audio, and streamed video playback
- practical gateway connectivity patterns remain supported, including Tailscale-friendly usage
- Android 11+ compatibility baseline remains unchanged in this fork

Important notes:

- this is still an early `Pre-release`
- this is not an official OpenClaw build
- the latest publicly published release may lag behind this development baseline until the next signed APK is cut

Install:

- download the attached APK
- allow installation from this source if Android prompts for it
- launch `ClawChat2`

## 中文

这是 ClawChat2 当前 Android 基线的继续迭代版本，也用于准备下一次公开构建。

ClawChat2 是一个非官方 OpenClaw Android 分叉，目标是在 Android 上以更简单、更直接、聊天优先的方式与 OpenClaw agent 对话。

本版本重点包括：

- 联系人、会话与聊天历史现在以 `openclaw-webchat` 为主数据源
- 联系人界面已补齐 agent 头像同步，并优化了联系人列表展示
- agent 图片、音频、流式视频等媒体能力仍是本分叉重点
- 继续保留更贴近实际使用的网关连接方式，包括对 Tailscale 场景的友好支持
- 本分叉仍以 Android 11+ 作为兼容基线

重要说明：

- 当前版本仍属于早期 `Pre-release`
- 这不是官方 OpenClaw 构建版本
- 在下一次签名 APK 发布前，公开仓库中的已发布版本可能仍落后于当前开发基线

安装方式：

- 下载附带的 APK
- 如果 Android 提示，请允许从当前来源安装
- 启动 `ClawChat2`
