# ClawChat2 v0.2.5 Release Notes

## English

This release promotes the screenshot-aligned chat palette to the new mainline baseline and prepares that version for the public repository.

ClawChat2 remains an unofficial OpenClaw Android fork centered on a simple, direct, chat-first experience for talking with OpenClaw agents.

Highlights in this build:

- the `codex/ui-palette-pass` chat visual work is now merged into `main`, making the new screenshot-aligned chat palette the ongoing development baseline
- light and dark mode chat surfaces were tightened against the release mocks, with quieter assistant surfaces and stronger separation between transcript layers
- user reply bubbles now use a solid green fill with dark text in both light and dark mode for better readability
- the onboarding home screen now explicitly tells users to first give the `claw-webchat` server-install guide to their agent or operator before continuing setup
- the repository homepage now carries the same server-side setup instruction and links directly to the `claw-webchat` install guide

Important notes:

- this is still an early `Pre-release`
- this is not an official OpenClaw build
- `openclaw-webchat` remains the source of truth for contacts and chat history
- before setup, users should ask their agent/operator to follow `https://github.com/memphislee09-source/claw-webchat/blob/main/docs/AGENT_INSTALL_NETWORK.md`

Install:

- download the attached APK
- allow installation from this source if Android prompts for it
- launch `ClawChat2`

## 中文

这个版本将按参考图收口后的聊天配色正式提升为新的主线基线，并以此作为公开仓库的升级版本。

ClawChat2 仍然是一个非官方 OpenClaw Android 分叉，目标是在 Android 上以更简单、更直接、聊天优先的方式与 OpenClaw agent 对话。

本版本重点包括：

- `codex/ui-palette-pass` 上完成的聊天视觉收口现已合并进 `main`，成为后续持续开发的新基线
- 浅色和深色模式下的聊天表层关系已继续按参考图收紧，assistant 容器更克制、对话层级更清晰
- 用户回复气泡现在在浅色和深色模式下都统一使用实心绿色底配深色文字，可读性更好
- app 安装后进入的 onboarding 首页现在会明确提示用户：继续设置前，先把 `claw-webchat` 的服务端安装文档发给 agent 或 operator
- 项目 README 首页也同步加入了相同提示，并直接链接到 `claw-webchat` 安装文档

重要说明：

- 当前版本仍属于早期 `Pre-release`
- 这不是官方 OpenClaw 构建版本
- 联系人与聊天历史的服务端真源仍然是 `openclaw-webchat`
- 开始设置前，应先让 agent/operator 按照这个文档完成 `claw-webchat` 服务端安装：
  `https://github.com/memphislee09-source/claw-webchat/blob/main/docs/AGENT_INSTALL_NETWORK.md`

安装方式：

- 下载附带的 APK
- 如果 Android 提示，请允许从当前来源安装
- 启动 `ClawChat2`
