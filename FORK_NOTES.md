# Fork Notes

## English

### Origin

This repository is derived from the official OpenClaw Android client located in `openclaw/openclaw -> apps/android`.

### Licensing And Attribution

- The upstream OpenClaw project is MIT-licensed.
- This fork preserves the upstream MIT license in [LICENSE](LICENSE).
- Third-party notices remain under [THIRD_PARTY_LICENSES](THIRD_PARTY_LICENSES).
- Original upstream authors retain copyright in the portions of code originated from the upstream project.

### Project Identity

- This repository is an independent fork.
- It is not an official OpenClaw repository.
- It is not published by, maintained by, or endorsed by the OpenClaw maintainers.

### Current Fork Direction

This fork is intentionally narrower than the general upstream Android client. Its current focus is:

- direct chat-first Android UX
- improved agent media receive/render behavior
- practical fullscreen media playback stability
- simple access patterns including Tailscale-friendly usage

### Early-State Warning

This project is still early. Product naming, branding, public packaging, and upstream strategy may change as the fork matures.
