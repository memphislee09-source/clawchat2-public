# ClawChat2 v0.2.1 Release Notes

## English

This release is the first public Android APK for ClawChat2.

ClawChat2 is an unofficial OpenClaw Android fork focused on a simple, direct, chat-first experience for talking with OpenClaw agents.

Highlights in this build:

- chat-first Android experience for direct agent conversations
- stronger support for agent-sent image, audio, and video messages
- stable fullscreen media handling for image and video viewing
- practical gateway connectivity patterns, including Tailscale-friendly usage
- Android 11+ compatibility baseline in this fork

Important notes:

- this is an early `Pre-release`
- this is not an official OpenClaw build
- compatibility and UX may still change quickly

Install:

- download the attached APK
- allow installation from this source if Android prompts for it
- launch `ClawChat2`

## 中文

这是 ClawChat2 首个公开 Android APK 版本。

ClawChat2 是一个非官方 OpenClaw Android 分叉，目标是在 Android 上以更简单、更直接、聊天优先的方式与 OpenClaw agent 对话。

本版本重点包括：

- 聊天优先的 Android 体验，面向与 agent 的直接对话
- 更强的 agent 图片、音频、视频消息接收能力
- 更稳定的图片与视频全屏媒体体验
- 更贴近实际使用的网关连接方式，包括对 Tailscale 场景的友好支持
- 本分叉以 Android 11+ 作为兼容基线

重要说明：

- 当前版本属于早期 `Pre-release`
- 这不是官方 OpenClaw 构建版本
- 兼容性与交互体验仍可能继续快速调整

安装方式：

- 下载附带的 APK
- 如果 Android 提示，请允许从当前来源安装
- 启动 `ClawChat2`
