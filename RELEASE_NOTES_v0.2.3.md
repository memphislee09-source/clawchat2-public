# ClawChat2 v0.2.3 Release Notes

## English

This release advances the current ClawChat2 Android baseline and sets the next ongoing development checkpoint.

ClawChat2 remains an unofficial OpenClaw Android fork focused on a simple, direct, chat-first experience for talking with OpenClaw agents.

Highlights in this build:

- contacts keep their synced agent avatars and now share a cleaner continuous-list interaction baseline with chat
- chat message presentation has been simplified to a spacing-first layout with no extra bubble borders
- user-side message labels now sync from `openclaw-webchat` settings, so the in-app chat header can match the webchat “me” name
- the user display-name sync now refreshes on the same cadence as contacts: app startup and explicit contacts pull-to-refresh
- agent-sent image, audio, and streamed video playback remain intact on the current Android baseline

Important notes:

- this is still an early `Pre-release`
- this is not an official OpenClaw build
- the latest publicly published release may still lag behind this development baseline until the next signed APK is cut

Install:

- download the attached APK
- allow installation from this source if Android prompts for it
- launch `ClawChat2`

## 中文

这是 ClawChat2 当前 Android 基线的进一步迭代版本，也作为下一阶段持续开发的新基点。

ClawChat2 仍然是一个非官方 OpenClaw Android 分叉，目标是在 Android 上以更简单、更直接、聊天优先的方式与 OpenClaw agent 对话。

本版本重点包括：

- 联系人页继续保留同步后的 agent 头像，并与聊天页一起收敛到更轻的连续列表交互风格
- 聊天消息展示已简化为以留白分组为主的布局，不再增加额外气泡边框
- 用户侧消息标题现在会同步 `openclaw-webchat` 设置中的“我”名称，让 app 对话页与 webchat 保持一致
- 用户名称同步的刷新频率与联系人一致：app 启动时一次，联系人页手动下拉刷新时一次
- agent 图片、音频、流式视频等媒体能力在当前 Android 基线上继续保持可用

重要说明：

- 当前版本仍属于早期 `Pre-release`
- 这不是官方 OpenClaw 构建版本
- 在下一次签名 APK 发布前，公开仓库中的已发布版本可能仍落后于当前开发基线

安装方式：

- 下载附带的 APK
- 如果 Android 提示，请允许从当前来源安装
- 启动 `ClawChat2`
